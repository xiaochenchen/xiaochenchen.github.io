Tags: blog, midnight commander, notes

How to use external editor to edit and view files?

You can change this in Options menu, by pressing F9. Select Configuration menu item, which will display Configure Options dialog. You just need to deselect the check box next to Use Internal Edit and Use Internal View.

How to change the Enter key behavior to view file instead of executing it?

Press F9 –> Select Command menu –> Select Edit extension file menu item. This will display the extension file. Go to the bottom of the file and change the value of open and view paramter values as shown below.

[Leetcode] Longest Consecutive Sequence

Published on: Mar 30th, 2014
Tags: array, coding, leetcode

Algorithm:

Store all elements in an HashSet H. Start scaning the array, for each element i in the array, if i is in H, check whether i-1 is in H, if yes, remove i-1 from H, continue to check i – 2 and not found in H. Do the same thing from i to its right. Keep track of current max length. Return max length at the end.
